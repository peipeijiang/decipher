# TikTok爆款视频分析平台 - 产品说明文档

**ViralLens** 是一个AI驱动的TikTok内容营销智能分析平台，帮助创作者和电商运营快速拆解爆款视频的成功密码，自动生成可复用的创意方案。

## 核心价值主张

我们解决的问题：内容创作者看到爆款视频却不知道如何复刻，电商运营缺乏数据驱动的视频策略。

**ViralLens的答案：** 上传任何TikTok爆款视频，系统在2-3分钟内自动拆解其营销策略、分镜结构、情感节奏，并生成可直接用于拍摄的创意提示词和产品改写方案。

## 产品功能模块

### 1. 视频智能分析引擎
- 自动提取视频关键帧（6帧均匀分布）
- 本地Whisper语音识别（支持中文）
- 多模型视觉分析（支持OpenAI、Claude、豆包、MiniMax等）
- 输出：分镜场景、内容钩子、情感共鸣点、复制建议

### 2. 创意改写工作台
- 基于爆款视频结构生成4个可复刻的创意角度
- 保留原视频的节奏骨架，替换产品内容
- 自动生成英文视频拍摄提示词（包含镜头语言、灯光、音效、演员描述）
- 支持UGC、专业广告、生活方式三种风格

### 3. 产品视频生成工作流
- 输入产品链接，自动爬取商品信息、图片、描述
- AI生成10个视频创意提示词（支持多种模板：GRWM、开箱、对比、教程等）
- 一键生成产品宣传图和视频（集成Laozhang图像生成API）
- 支持提示词微调、重新生成、批量操作

### 4. 产品说明书自动生成
- 从产品信息自动生成专业的产品使用说明书图片
- 包含产品概览、使用步骤、注意事项、常见问题
- 可用于电商平台、社交媒体营销

### 5. 配置管理中心
- 灵活切换AI模型（视觉分析和综合分析独立配置）
- 支持8+种AI模型供应商
- 模板库管理（视频风格、Hook策略可自定义）
- 用户权限和API配额管理
