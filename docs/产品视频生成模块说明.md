# 产品视频生成模块 - 详细功能说明

## 一、功能概述

产品视频生成模块是 ViralLens 平台的核心功能之一，实现了**从商品链接到成品视频**的全自动化流水线。用户只需输入一个商品链接，系统自动完成产品分析、文档生成、说明书制作、视频脚本创作、参考图生成和视频合成的完整流程。

### 泰式剧情广告模板详解

泰式广告被公认为全球最具创意和传播力的广告形式之一，其核心精髓：

```
┌─────────────────────────────────────────┐
│         泰式广告叙事结构                   │
├─────────────────────────────────────────┤
│                                         │
│  0%────────────80%──────────────100%    │
│  │              │                │      │
│  │  纯故事叙述   │  产品揭示反转   │      │
│  │  不提产品     │  解决一切       │      │
│  │              │                │      │
│  情绪渐进铺垫 → 情绪顶点反转 → 温馨收尾  │
│                                         │
└─────────────────────────────────────────┘
```

**关键特征：**
- 前80%完全是日常生活故事，观众不知道这是广告
- 情绪要么从搞笑到催泪，要么从催泪到反转搞笑
- 产品出场 = 情节反转点（意料之外、情理之中）
- 演技夸张、表情丰富、生活化场景

## 七、核心亮点

### 🎯 亮点 1：两阶段生成，用户完全掌控

不同于传统的"一键生成"，系统分为自动分析和手动创作两个阶段。用户可以：
- 先查看产品分析结果和说明书
- 再决定用什么风格生成
- 每个变体独立配置，灵活调整

### 🎨 亮点 2：多维度模板组合

```
视频风格（9种）× 开场白策略（10种）× 图片布局（6种）
= 540 种创意组合
```

每个变体可以独立选择不同的风格+开场白组合，一次性产出多种风格的创意方案。

### 🔄 亮点 3：非破坏性微调

提供三种修改方式：
1. **切换模板** → 重新生成整个提示词
2. **微调** → 用自然语言描述修改意图，AI 在现有基础上定向调整
3. **手动编辑** → 直接编辑文本

### 🖼️ 亮点 4：产品一致性保证

通过多层机制确保生成的图片/视频中产品外观正确：
1. AI 视觉识别提取产品外观描述
2. AI 从说明书提取产品使用方式
3. 产品实物照片作为参考图上传
4. 说明书作为额外参考图（Seedance）
5. Prompt 中注入详细的产品描述约束

### 📐 亮点 5：分镜可视化

选择宫格布局后，AI 会将视频脚本转换为分镜图：
- 每个面板描述关键画面
- 保持产品外观一致
- 可以直接用于拍摄团队参考

### ⚡ 亮点 6：模板可配置

所有模板（视频风格、开场白、图片布局）都可以在管理后台自由配置：
- 查看/编辑/新增/删除
- 启用/禁用控制
- 内置模板受保护不可删除
- 支持自定义扩展

## 八、典型使用场景

### 场景 A：跨境电商新品推广

```
1. 输入 Amazon 商品链接
2. 系统自动分析产品（2-3分钟）
3. 查看说明书，确认产品理解正确
4. 选择"开箱测评"+"即时好奇"生成提示词
5. 微调："加入更多产品特写镜头"
6. 生成6宫格分镜图 → 发给拍摄团队
7. 生成AI视频 → 直接用于投流测试
```

### 场景 B：品牌营销内容矩阵

```
1. 同一产品，一次生成10个不同风格的视频脚本
2. 变体1: GRWM + 痛点解决
3. 变体2: 教程 + 你一直做错了
4. 变体3: 泰式剧情 + 社交证明
5. 变体4: ASMR + 即时好奇
6. ...每个都生成图片和视频
7. A/B测试选出最佳创意方向
```

### 场景 C：TikTok 矩阵号运营

```
1. 批量导入产品链接
2. 每个产品自动生成10个创意
3. 批量生成图片和视频
4. 直接发布到多个账号
5. 根据数据反馈，微调表现好的创意
```
